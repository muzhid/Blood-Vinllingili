Managing Access, Passwords, and Users.

1. Getting Admin Access

If you need access to the dashboard for the first time:

  1. Go to the Telegram Admin Group.
  2. Type the command: /admin_access
  3. The bot will send you a Temporary Username & Password.
  4. Go to the Login Page and use these credentials.
Security: Change your password immediately after your first login.

2. Resetting Passwords

Method A: Forgot Password (via Telegram)

If you are locked out:

  1. Go to the Telegram Admin Group.
  2. Type: /reset_password
  3. The bot will reset your password to a temporary one and send it to you.

Method B: Changing Password (via Dashboard)

If you are already logged in:

  1. Go to Settings > Update Password.
  2. Enter your new password and confirm.

3. Removing Users

Users cannot delete themselves. They must request removal.

  1. When a user types "Remove me" in the bot, a request is sent to the Admin Group.
  2. You will see a message with [Approve Removal] and [Reject] buttons.
  3. Click [Approve Removal] to permanently delete their data.
You can also manually delete users from the Admin Users Table in the dashboard.
