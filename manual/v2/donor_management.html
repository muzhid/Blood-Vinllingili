Adding and Editing Donors via AI & Graphic Interface.

1. Adding Donors via ID Scan (AI)

The fastest way to register a new donor is by scanning their Maldives ID Card.

  1. Go to the Telegram Admin Group.
  2. Upload a Photo of the ID Card.
  3. In the caption (or a reply), type /scan (optional, bot usually auto-detects).
  4. The AI will extract: Name, ID Number, Address, Sex, Date of Birth.
  5. The bot will ask you to confirm adding them to the database.
Common Issues:
  • Glare/Reflection: Ensure the text is readable.
  • Blur: Hold the camera steady.
  • Wrong Side: Upload the front side with the Name/ID.

2. Adding Donors Manually (GUI)

If ID scanning fails or you need to add someone without a card:

  1. Login to the Dashboard.
  2. Click the + Add User button (top right).
  3. Fill in the required details (Name, Phone, Blood Type).
  4. Click Save.

3. Editing & Deleting

To update a donor's info (e.g. they moved island or changed phone):

  • Find them in the User Table.
  • Click the Edit (Pencil Icon).
  • Update the fields and click Save Changes.

To Delete a user, click the Trash icon in the Edit menu. Warning: This is permanent.
