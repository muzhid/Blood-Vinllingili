For Nurses & Medical Staff.

1. Accessing the Portal

Login to the web dashboard using the credentials provided by your IT admin.

You will see the User Table containing all registered donors.

2. Finding a Donor

Use the Search Bar at the top of the table to find a donor by:

  • Name
  • Phone Number
  • ID Card Number

You can also sort the table by Blood Type.

3. Marking a Donation

When a donor successfully donates blood:

  1. Find their row in the table.
  2. Click the Mark Donated button.
  3. Select the Date of Donation from the calendar popup.
  4. Click Confirm.
System Logic: The system will automatically calculate their Next Eligible Date (3 months from today) and mark their status accordingly.

4. Eligibility Status

The table shows color-coded indicators:

  • Eligible - Can donate now.
  • Wait - Recently donated. Check "Next Eligible" column.
